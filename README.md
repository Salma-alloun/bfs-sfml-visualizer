📌 BFS Ultra Optimisé avec Visualisation SFML
🧠 Description

Ce projet est une implémentation ultra optimisée de l’algorithme BFS (Breadth-First Search) en C++ avec une visualisation graphique interactive grâce à SFML.

L’application permet de :

Visualiser l’exploration d’une grille en temps réel
Comparer une exécution animée vs ultra rapide (~0.2 ms)
Observer les performances détaillées de l’algorithme
🚀 Fonctionnalités
🔍 Algorithme
BFS optimisé avec structures mémoire efficaces
File circulaire performante
Distances et parents stockés efficacement
Reconstruction rapide du plus court chemin
🎮 Interface interactive
Grille dynamique (15x15)
Placement de :
🔴 Source
🟡 Destination
⚫ Obstacles
Visualisation des états :
🟢 Visité
🔵 Chemin le plus court
🟣 File d’attente
🟠 Nœud courant
⚡ Modes d’exécution
🎬 Mode animation (pas à pas)
⚡ Mode ultra rapide (~0.2 ms)
📊 Statistiques en temps réel
Temps de recherche pur
Temps avec animation
Temps total
Nombre de cellules visitées
Longueur du chemin
Taille de la file

🛠️ Technologies utilisées
C++
SFML (Simple and Fast Multimedia Library)
STL (vector, queue, chrono, etc.)

💡 Objectif pédagogique

Ce projet permet de :

Comprendre en profondeur BFS
Visualiser le parcours en largeur
Apprendre l’optimisation bas niveau en C++
Manipuler SFML pour des interfaces graphiques
